# Alveolar lineage divergence during targeted therapy in lung adenocarcinoma

This repository contains code related to the manuscript:

> Zatzman, M., Quintanal-Villalonga, A. _et al_, Alveolar lineage divergence during targeted therapy in lung adenocarcinoma.

## Data Availablility

Raw sequencing data generated in this project has been deposited to the GEO Expression Omnibus under accession [GSE292469](link). 

Cohort, epithelial, and tumor histotime scRNA data objects can be interactively visualized and downloaded from [CELLxGENE](https://cellxgene.cziscience.com/collections/df8ef04c-7a87-49d4-a127-f8d653d89d91).
